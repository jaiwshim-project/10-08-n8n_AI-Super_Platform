# S1.A1.L1 — Requirements (요구사항)

## 프로젝트 목표
n8n 기반 AI 워크플로우 자동화 플랫폼의 소개 및 관리 웹 UI

## 핵심 요구사항

### 기능 요구사항
1. **메인 랜딩** — 플랫폼 개요, 주요 기능 소개, CTA
2. **아키텍처 뷰** — 시스템 구성도, 탭 기반 상세 뷰
3. **사용 매뉴얼** — 단계별 가이드, 검색 가능한 문서
4. **AI 챗봇** — Gemini API 연동, 실시간 대화 인터페이스
5. **대시보드** — KPI 카드, 워크플로우 현황 모니터링
6. **에이전트 목록** — AI 에이전트 카탈로그, 검색/필터

### 비기능 요구사항
- **스택**: Vanilla HTML/CSS/JS (프레임워크 없음)
- **반응형**: 모바일/태블릿/데스크탑 지원
- **접근성**: WCAG 2.1 AA (ARIA, 키보드 내비게이션)
- **성능**: 빠른 로딩 (CDN 최소화, 이미지 최적화)
- **테마**: 다크 블루 (#050c1a 기반) + 블루 카드 (#1e5ab5~#2469c8)
