n8n 매뉴얼 & 가이드

n8n 자동화 플랫폼의 핵심 노드, 워크플로우 설계, 사업 적용 방법을 단계별로 설명합니다. AI와 자동화를 결합하여 업무 생산성을 10배 높이세요.

⚡
n8n 소개

n8n은 코드 없이도 강력한 자동화 워크플로를 구축할 수 있는 오픈소스 자동화 플랫폼입니다.

n8n이란?

Node(노드) + N(엔드리스)의 합성어로, 수백 개의 서비스를 연결하는 노드 기반 자동화 도구입니다. 시각적 편집기로 복잡한 비즈니스 로직을 구현할 수 있습니다.

  • 400+ 내장 통합(Integrations)
  • AI 에이전트 노드 내장
  • 자체 호스팅 가능 (데이터 보안)
  • JavaScript/Python 코드 노드

핵심 개념

  • 워크플로(Workflow): 노드가 연결된 자동화 흐름
  • 트리거(Trigger): 워크플로 시작 조건
  • 노드(Node): 각 단계의 작업 단위
  • 크레덴셜(Credential): API 인증 정보
  • 실행(Execution): 워크플로 한 번의 실행

핵심 노드 8종

n8n의 8가지 핵심 노드를 마스터하면 대부분의 자동화 워크플로를 구현할 수 있습니다.

🌐
Webhook
Trigger
HTTP POST/GET 요청을 받아 워크플로를 시작합니다. 외부 서비스에서 데이터를 실시간으로 수신할 때 사용합니다.
파라미터: Path Method Response Mode
📡
HTTP Request
Action
REST API를 호출하여 데이터를 가져오거나 전송합니다. 인증 방식(API Key, OAuth, Basic)을 지원합니다.
파라미터: URL Method Headers Body
🤖
AI Agent
AI
LangChain 기반 AI 에이전트를 실행합니다. GPT-4, Gemini, Claude 등 LLM을 연결하고 도구(Tools)를 제공할 수 있습니다.
파라미터: Agent Type Chat Model Memory Tools
💻
Code
Logic
JavaScript 또는 Python 코드를 직접 실행합니다. 복잡한 데이터 변환, 계산, 조건 처리를 구현할 수 있습니다.
파라미터: Language Code Mode
🔀
IF
Logic
조건에 따라 워크플로를 분기합니다. 참(True)/거짓(False) 두 경로로 데이터를 다르게 처리할 수 있습니다.
파라미터: Conditions Combine
📊
Google Sheets
Service
Google Sheets에서 데이터를 읽고 쓸 수 있습니다. 업무 데이터베이스, 보고서 자동화에 활용합니다.
파라미터: Spreadsheet ID Sheet Name Operation
💬
Slack / 메신저
Notify
Slack, Discord, Telegram 등 메신저로 알림을 전송합니다. 리치 메시지, 첨부파일, 버튼 포함 가능합니다.
파라미터: Channel Text Blocks
⏰
Schedule Trigger
Trigger
Cron 표현식으로 정기적으로 워크플로를 실행합니다. 매일, 매주, 매월 등 원하는 주기로 자동 실행합니다.
파라미터: Rule Timezone

자주 묻는 질문 (FAQ)
n8n은 무료인가요?
▼

n8n의 Community Edition은 완전 무료 오픈소스입니다. 자체 서버(Docker, VPS)에 설치하여 제한 없이 사용할 수 있습니다. n8n Cloud(관리형 서비스)는 월 $20~부터 유료입니다. 처음에는 n8n Cloud의 무료 트라이얼(14일)로 시작하고, 규모가 커지면 자체 호스팅으로 전환하는 것을 권장합니다.

코딩을 몰라도 사용할 수 있나요?
▼

기본적인 워크플로우는 코딩 없이 드래그앤드롭으로 구성할 수 있습니다. 그러나 복잡한 데이터 처리, 조건 로직, API 커스터마이징에는 JavaScript 기초 지식이 필요합니다. AI 에이전트가 n8n 코드 작성을 도와주므로 완전 비개발자도 점진적으로 학습하며 활용 가능합니다.

Zapier 대비 n8n의 가장 큰 장점은?
▼

① 비용: 자체 호스팅 시 무제한 실행 무료 (Zapier는 실행 횟수당 과금), ② AI 통합: LangChain 기반 AI 에이전트가 내장되어 복잡한 AI 워크플로 구현 가능, ③ 데이터 보안: 모든 데이터가 자사 서버에만 저장됨, ④ 유연성: JavaScript/Python 코드 노드로 제한 없는 로직 구현 가능합니다.

n8n으로 어떤 AI 모델을 사용할 수 있나요?
▼

n8n은 LangChain을 통해 OpenAI(GPT-4o, o1), Anthropic(Claude 3.5), Google(Gemini 1.5 Pro), Meta(Llama 3), Mistral 등 주요 LLM을 모두 지원합니다. 자체 호스팅 Ollama를 통해 로컬 모델도 사용 가능합니다. AI Agent 노드에서 원하는 모델을 선택하고 API 키만 설정하면 즉시 사용할 수 있습니다.

n8n 서버는 어디에 설치하나요?
▼

권장 설치 환경: ① AWS EC2 / GCP / Azure 클라우드 서버 (월 $10~20 t3.small), ② DigitalOcean Droplet (월 $6~), ③ 로컬 Docker (개발용). 프로덕션 환경에는 최소 2GB RAM, PostgreSQL DB, Nginx 리버스 프록시를 권장합니다. 공식 Docker Compose 파일로 30분 이내 설치 가능합니다.
