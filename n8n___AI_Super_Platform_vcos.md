## 📋 PROJECT INTENT

대표님이 올리신 문서는 **n8n 자동화 플랫폼을 활용하는 방법을 정리한 “자동화 설계 치트시트”**입니다.

1. n8n 이름의 의미

n8n은 다음 의미입니다.

“nodemation”

즉

node + automation

입니다.

여기서

node = 작업 하나
automation = 자동화

그래서

n8n은

여러 개의 작업(node)을 연결하여 자동으로 실행하는 시스템

이라는 의미입니다.

예를 들어

고객 문의
→ AI 분석
→ CRM 저장
→ 이메일 발송
→ Slack 알림

이 모든 과정을 자동으로 연결합니다.

2. n8n이 무엇인지 한 문장으로 설명

가장 쉬운 설명은 이것입니다.

“AI와 앱을 연결해서 일을 자동으로 처리하는 자동화 플랫폼”

입니다.
